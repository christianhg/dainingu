Prerequisites
-------------------
Be sure to have the following prerequisites installed.

- Node (0.10.x)
- NPM (1.4.x)
- Gulp (3.8.10)
- ...

Installation
-------------------
Install application using npm.

```
$ npm install
```

Run
-------------------
Run application using Gulp.

```
$ gulp
```
